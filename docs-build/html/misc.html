get_timer(pin, index=0)¶

A method to get a timer connected to a pin.

Parameters:
  • pin – Pin that timer should be connected to
  • index – the index of the timer on that pin (for the case of multiple timers connected to one pin).
Returns:

The timer number and channel associated with that pin.
