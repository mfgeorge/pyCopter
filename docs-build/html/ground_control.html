ground_control module¶

Authors

Michael George
Oscar Ruiz

Module for hosting connections to ground control on the LoPy The scheme for incoming commands to be parsed is as follows:

incoming = b'p_val1;r_val2;y_val3;t_val4'

where incoming is then parsed and the letters p, r, y, and t indicate to stick the val that follows into the pitch, roll, yaw, or thrust set-point respectively. Commands can also be added dynamically during run-time or start up using the add_command() method.

add_command(command_letter, command_callback)¶

Method for adding a command to the command_dict so that a received command letter has a callback it is mapped to.

Parameters:
  • command_letter – the command letter that will be received from ground control
  • command_callback – the callback associated with the letter

Warning

command_callback needs to take exactly one parameter to work with the parsing scheme.
